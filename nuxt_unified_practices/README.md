# Единый Nuxt-проект по практическим работам

Проект объединяет работы из чата:

1. **Практика 1 — Кваккер / блогеры**
   - перенос HTML-вёрстки на Nuxt;
   - получение данных из MockAPI;
   - вывод постов;
   - получение случайной шутки;
   - форма создания пользователя;
   - админ-панель с randomuser.me.

2. **Практика 2 — Выбор пути / routeRules**
   - `/` и `/index` → `prerender: true`;
   - `/user_page` → `swr: true`;
   - `/recommendations` → `swr: 3600`;
   - `/admin` → `ssr: false`;
   - `/login` → `redirect: '/enter'`.

3. **Практика 3 — Реактивный iPhone**
   - вкладки Overview / Tech Specs / Pricing / Software;
   - реактивный выбор памяти;
   - выбор языка;
   - светлая/тёмная тема;
   - composables и utils.

4. **Практика 4 — Дополнительная реактивная страница iPhone**
   - отдельный маршрут `/catalog/iphone-reactive/16`;
   - реактивные вкладки, память, язык и тема;
   - логика вкладок и состояния вынесена в composables;
   - дополнительные функции товара вынесены в `utils/productContent.ts`.

5. **Практика 5 — Социальная сеть**
   - отдельный маршрут `/social-network`;
   - профиль Егора Ангелова;
   - друзья, новости, форма создания поста;
   - динамический переход на страницу друга внутри страницы.

6. **Практика 6 — Реактивная социальная сеть**
   - отдельный маршрут `/social-network-reactive`;
   - счётчики друзей, постов пользователя и постов в ленте;
   - поиск друзей через `computed`;
   - реактивные лайки для постов;
   - пустые состояния для профиля без постов и поиска без результатов.

## Запуск

```bash
npm install
npm run dev
```

## Основные маршруты

- `/` — главная навигационная страница;
- `/practice-1` — социальная сеть блогеров;
- `/practice-2` — объяснение routeRules;
- `/catalog/iphone/16` — реактивная страница товара;
- `/catalog/iphone-reactive/16` — дополнительная реактивная страница iPhone;
- `/social-network` — страница социальной сети;
- `/social-network-reactive` — соцсеть со счётчиками, поиском, лайками и пустыми состояниями;
- `/admin` — CSR-админка;
- `/user_page` — SWR-страница пользователя;
- `/recommendations` — рекомендации с кешем 3600 секунд;
- `/login` — редирект на `/enter`.
