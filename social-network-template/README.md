# Social Network Lesson Project

Это версия проекта только по занятию **«Социальная сеть»**, начиная с раздела **«1. Профиль пользователя»**.

Внутри есть:

- профиль пользователя;
- загрузка пользователя через GET-запрос;
- загрузка постов через GET-запрос;
- создание поста через POST-запрос;
- вывод массивов через `v-for`;
- страницы `/`, `/friends`, `/user/:username`, `/feed` через Vue Router;
- страница друзей;
- динамический маршрут `/user/:username`;
- страница новостей `/feed`;
- конспект занятия в `docs/social-network-lesson.md`.

Сервер использует учебные JSON-файлы из папки `json/`, поэтому MongoDB для запуска этой версии не нужна.

## Запуск

Установить зависимости:

```bash
npm install
```

Запустить сервер:

```bash
npm run start
```

Запустить клиент во втором терминале:

```bash
npm run dev
```

Сервер работает на:

```txt
http://localhost:3005
```

Клиент работает на:

```txt
http://localhost:5173
```
